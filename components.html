Components in React.js

Components are the building blocks of a React application. They allow you to split the user interface into small, reusable pieces, making it easier to manage and maintain your code.

Types of Components

In React, there are two types of components:

  • Functional Components: Functional components are defined as JavaScript functions and return JSX to describe the user interface. They are simpler and more lightweight compared to class components.
  • Class Components: Class components are defined as JavaScript classes that extend the `React.Component` base class. They have additional features such as lifecycle methods and internal state management.

Example: Functional Component

Here's an example of a functional component:


import React from 'react';

const Greeting = (props) => {
  return (
    <div>
      <h2>Hello, {props.name}!</h2>
      <p>Welcome to our website.</p>
    </div>
  );
};

export default Greeting;

In the above example, the `Greeting` component is defined as a JavaScript arrow function that accepts `props` as a parameter. It returns JSX to render a greeting message and a welcome message.

Example: Class Component

Here's an equivalent example of a class component:


import React from 'react';

class Greeting extends React.Component {
  render() {
    return (
      <div>
        <h2>Hello, {this.props.name}!</h2>
        <p>Welcome to our website.</p>
      </div>
    );
  }
}

export default Greeting;

In this example, the `Greeting` component is defined as a class that extends the `React.Component` base class. The `render` method returns JSX to render the greeting message and the welcome message.

Conclusion

Components are a fundamental part of React, allowing you to build reusable and modular user interfaces. Whether you choose functional components or class components, React provides a flexible and powerful component-based architecture to develop your application.
