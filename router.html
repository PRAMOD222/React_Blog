Router in React.js

React Router is a popular library that allows you to handle routing in React applications. It provides a way to navigate between different components and render them based on the URL.

Why Use React Router?

React Router helps in building single-page applications (SPAs) with multiple views or pages. It enables the creation of dynamic and responsive user interfaces by managing the application's routing and rendering components based on the current URL.

Using React Router

React Router provides different components to handle routing in your application:

  • BrowserRouter: Handles routing using HTML5 history API, providing clean URLs without page refresh.
  • Switch: Renders the first child `` or `` that matches the current URL.
  • Route: Renders a component based on the URL path.
  • Link: Provides a navigation link to navigate between different routes.

Example: Using React Router

Here's an example of using React Router to handle routing in a React application:


import React from 'react';
import { BrowserRouter as Router, Switch, Route, Link } from 'react-router-dom';

import Home from './components/Home';
import About from './components/About';
import Contact from './components/Contact';

function App() {
  return (
    <Router>
      <div>
        <nav>
          <ul>
            <li>
              <Link to="/">Home</Link>
            </li>
            <li>
              <Link to="/about">About</Link>
            </li>
            <li>
              <Link to="/contact">Contact</Link>
            </li>
          </ul>
        </nav>

        <Switch>
          <Route exact path="/" component={Home} />
          <Route path="/about" component={About} />
          <Route path="/contact" component={Contact} />
        </Switch>
      </div>
    </Router>
  );
}

export default App;

In the above example, the `App` component sets up the router using the `BrowserRouter` component from React Router. It defines different routes using the `Route` component, and the `Link` component provides navigation links for the routes.

When a user clicks on a navigation link, the `Switch` component renders the corresponding component based on the current URL path.

Conclusion

React Router is a powerful library that simplifies the routing process in React applications. It enables the creation of dynamic and interactive user interfaces by handling navigation between different components based on the URL.
